### Variables

 - No unused variables
 - Default values are 0, false or empty
 - Strongly Types

```
var a int 32
a = 15

# Allocates 32 bytes of memory, probably 4 8 byte hex memory spots in ram for a, then adds value to it

0x0000000064
0x0000000065
0x0000000066
0x0000000067 | 15
```
